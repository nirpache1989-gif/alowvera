# Visual Asset Brief

## Direction

השפה הוויזואלית של האתר מבוססת על `Botanical Editorial Atlas`: שילוב של אור רך, שכבות נייר, מרקמים מינרליים, טון טבעי-אוצרותי, ותנועה רגועה.

## Asset Families

1. Hero panoramas
- 6-8 תמונות פנורמיות ביחס `16:9` ו-`21:9`.
- לכל תמונה צריך מוקד טבעי ברור בצד אחד, ואזור "שקט" לטקסט בצד השני.
- לא להשתמש במראה stock. עדיף קומפוזיציות כמו סטודיו-מערכת, טבע מבוים, צמחים, זכוכית, קרמיקה, נוזלים, אריזות, אור בוקר/שקיעה.

2. Texture plates
- רקעי נייר, אבן, ערפל, סיבי צמח, זכוכית מטושטשת.
- שכבות עדינות להפרדות בין מקטעים וכרטיסים.

3. Packshot system
- צילום מוצר נקי על רקע בהיר חם.
- צל רך אחד, הילה קלה מאחור, ו-cutout אחיד.
- לא להשתמש בצללים דרמטיים מדי.

## Suggested Prompt Pattern

```text
Create a panoramic editorial wellness scene for a luxury Hebrew RTL e-commerce experience.
Style: botanical atlas, warm ivory, mineral green, clay, pollen gold.
Composition: asymmetrical, spacious, cinematic, tactile, not stock photography.
Include layered natural materials, subtle motion cues, soft daylight, refined atmosphere.
Leave intentional negative space for typography on the right side.
```

## Route-to-Asset Mapping

- Home hero: atlas-wide introduction
- Goals: energy / digestion / skin / balance / reset
- Routines: morning / evening / travel / recovery
- Brand panels: one key still-life per brand
- Journal: abstract texture-led thumbnails
